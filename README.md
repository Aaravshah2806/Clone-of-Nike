# Nike Website Desktop Clone

![Version](https://img.shields.io/badge/version-1.0.0-blue.svg)
![License](https://img.shields.io/badge/license-MIT-green.svg)

This project is a Windows Forms desktop application developed in C# that simulates a basic e-commerce website experience for Nike products. It allows users to navigate through various forms, showcasing product displays, specifically focusing on men's shoes, using local image resources.

## Features

- Simulated Nike product browsing interface.
- Multiple interactive forms for navigation and content display.
- Dedicated section for displaying men's shoes with associated images.
- Utilizes local image resources for product visuals and UI elements.
- Basic desktop application user experience for a product catalog.

## Installation

Step-by-step installation instructions:

```bash
# Clone the repository
git clone https://github.com/Aaravshah2806/Clone-of-Nike.git

# Navigate to project directory
cd Clone-of-Nike

# Open the solution in Visual Studio
# Double-click the 'NikeWebsite.sln' file or open it directly from Visual Studio.

# Build the solution
# In Visual Studio, go to 'Build' > 'Build Solution' to compile the project.
```

## Usage

Detailed usage instructions with examples:

```bash
# Run the application
# After building, press F5 or click the "Start" button (green play icon) in Visual Studio.
```
